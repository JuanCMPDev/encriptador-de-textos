# Encriptador de Texto | ORACLE-ONE

Una herramienta web para encriptar y desencriptar textos utilizando un conjunto de reglas específicas. Esta aplicación fue creada como parte del curso de ORACLE-ONE y está desarrollada con HTML5, CSS3 y JavaScript.

## Descripción

Esta aplicación permite a los usuarios encriptar y desencriptar texto usando las siguientes reglas de encriptación:

- La letra `e` se convierte en `enter`
- La letra `i` se convierte en `imes`
- La letra `a` se convierte en `ai`
- La letra `o` se convierte en `ober`
- La letra `u` se convierte en `ufat`

El proyecto incluye un navbar que se convierte en un menú hamburguesa en pantallas pequeñas, y proporciona una interfaz fácil de usar para interactuar con las funcionalidades de encriptación y desencriptación.

## Características

- **Encriptación y Desencriptación**: Convierte texto a encriptado y viceversa.
- **Interfaz de Usuario**: Navegación responsiva con un menú hamburguesa para pantallas pequeñas.
- **Estilos**: Diseño moderno y limpio usando CSS.

## Instalación

Para ejecutar la aplicación en tu entorno local, sigue estos pasos:

1. **Clona el Repositorio**

   ```bash
   git clone https://github.com/JuanCMPDev/encriptador-de-textos.git
   ```

2. **Navega al Directorio del Proyecto**

   ```bash
   cd encriptador-texto
   ```

3. **Abre los Archivos en tu Navegador**

   No se requiere un servidor local para esta aplicación, simplemente abre `index.html`, `encriptador.html` o `desencriptador.html` en tu navegador.

## Uso

1. **Encriptar Texto**

   - Dirígete a la página `encriptador.html`.
   - Introduce el texto que deseas encriptar en el campo de entrada.
   - Haz clic en el botón "Encriptar" para obtener el texto encriptado.

2. **Desencriptar Texto**

   - Dirígete a la página `desencriptador.html`.
   - Introduce el texto encriptado en el campo de entrada.
   - Haz clic en el botón "Desencriptar" para obtener el texto original.

## Estructura de Archivos

- **`index.html`**: Página principal de la aplicación.
- **`encriptador.html`**: Página para encriptar texto.
- **`desencriptador.html`**: Página para desencriptar texto.
- **`styles.css`**: Archivo de estilos para el diseño de la aplicación.
- **`scripts.js`**: Archivo de JavaScript para manejar la lógica de encriptación, desencriptación y el menú hamburguesa.
